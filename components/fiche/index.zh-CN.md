卡片。

## 何时使用

最基础的卡片容器，可承载文字、列表、图片、段落，常用于后台概览页面。

## API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| type | 卡片类型 | lite|normal | normal |
| title | 卡片标题 | string|ReactNode | 无 |
| extra | 卡片右上角的操作区域 | string|ReactNode | 无 |
| bottom | 卡片底部区域 | string|ReactNode | 无 |
| bodyStyle | 内容区域自定义样式 | object | 无 |
