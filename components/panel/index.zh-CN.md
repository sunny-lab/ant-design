面板。

## 何时使用

- 详情页显示。

## API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| id | 唯一标识 | string | |
| title | 标题 | string\|ReactNode | |
| description | 描述 | string\|ReactNode | |
| bordered | 是否显示边框 | boolean | true |
| scrollEl | 滚动条所在元素的Id或者其本身 | string\|HtmlElement | |
| time | 滚动动画时长(需要安装 @tweenjs/tween.js) | number | 500 |
